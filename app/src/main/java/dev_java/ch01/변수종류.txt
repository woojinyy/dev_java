원시형 변수
char, byte, short, int, long, float, double- 정수형,실수형 크기가 다르다,범위가 다르다, scope가 다르다
boolean(참거짓)-제어문에서 중요함,흐름을 다르게 할 때,분기문(if, switc)
호출하면 값이 나온다
원시형 변수에는 기본값이 있다.
원시형 변수를 초기화 할 때는 대입연산자를 사용한다
변수의 사용목적은 일괄처리에 있다.
